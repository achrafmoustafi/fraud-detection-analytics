# 🌍 Project Overview

This project analyzes financial transaction data to detect **fraudulent activities and suspicious patterns**.

The objective is to identify anomalies in transaction behavior using **machine learning models and data analytics techniques**.

The project combines **Python, anomaly detection algorithms, and Power BI dashboards** to support fraud monitoring and financial risk management.

---

# 🎯 Project Objectives

✔ Detect suspicious financial transactions  
✔ Identify anomaly patterns in transaction data  
✔ Build machine learning models for fraud detection  
✔ Visualize fraud indicators using interactive dashboards  

---

# 📊 Dashboard Features

The dashboard includes:

- Suspicious transaction monitoring
- Fraud risk indicators
- Transaction pattern analysis
- Anomaly detection visualization
- Interactive filters for fraud investigation
